# Enhanced Oil Recovery Method Selection System

## Project Overview
A machine learning-based system to predict and recommend the most suitable 
Enhanced Oil Recovery (EOR) method based on reservoir properties and conditions.

##  Objective
To help petroleum engineers select the optimal EOR method by analyzing 
reservoir data, enabling data-driven decision making for oil extraction.

##  Technologies Used
- Python
- Artificial Neural Network (ANN)
- Random Forest
- Scikit-learn, Pandas, NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Project Files
| File | Description |
| `Artificial neural network.ipynb` | ANN model implementation |
| `Random_forest.ipynb` | Random Forest model implementation |
| `eor dataset.zip` | Dataset used for training and testing |
| `ieee_journal_paper1.pdf` | Research paper |

## Models Used

### 1. Artificial Neural Network (ANN)
- Deep learning model trained on reservoir parameters
- Accuracy: 97.33%

### 2. Random Forest
- Ensemble learning method
- Prediction Accuracy: 1.0

##  How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Open `.ipynb` files in Jupyter Notebook or VS Code
4. Run all cells
